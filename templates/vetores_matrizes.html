Vetores e matrizes


Conceito
Vetores (listas) e matrizes (listas de listas) são estruturas de dados fundamentais na programação e na matemática, utilizadas para organizar e armazenar coleções de elementos do mesmo tipo.

  • Vetor (ou Array Unidimensional): É uma estrutura linear que armazena uma sequência de elementos em posições contíguas na memória. Cada elemento é acessado por um único índice que representa sua posição dentro da sequência. Imagine um vetor como uma única linha de caixas, onde cada caixa contém um valor e tem um número que indica sua posição.

  • Matriz (ou Array Bidimensional): É uma estrutura bidimensional que armazena elementos em linhas e colunas. Cada elemento é acessado por dois índices: um para a linha e outro para a coluna em que ele se encontra. Pense em uma matriz como uma tabela, onde cada célula contém um valor e é identificada por seu número de linha e número de coluna.

    • Generalizando, podemos ter arrays multidimensionais, que estendem o conceito para mais de duas dimensões (por exemplo, um cubo de dados seria um array tridimensional).


    Aplicação
    Vetores e matrizes são amplamente utilizados em diversas áreas da computação e da ciência, incluindo:

    Vetores:

  • Armazenamento de listas de itens: Nomes, notas, preços, etc.

  • Representação de dados sequenciais: Séries temporais, sequências de DNA.

  • Implementação de outras estruturas de dados: Pilhas, filas, listas encadeadas podem ser implementadas usando vetores como base.

  • Operações matemáticas e científicas: Representação de vetores em álgebra linear, armazenamento de coeficientes de polinômios.

  • Processamento de sinais e imagens: Armazenamento de amostras de áudio ou pixels de uma linha de uma imagem.


    • Matrizes:

  • Representação de tabelas de dados: Planilhas, bancos de dados.

  • Operações matemáticas e científicas: Álgebra linear (multiplicação de matrizes, resolução de sistemas de equações), transformações geométricas (rotação, escala, translação).

  • Processamento de imagens: Representação de imagens como grades de pixels (onde cada elemento da matriz representa a cor de um pixel).

  • Computação gráfica: Modelagem 3D, projeções.

  • Análise de dados: Representação de conjuntos de dados multivariados.

  • Inteligência Artificial e Machine Learning: Representação de pesos em redes neurais, matrizes de características

    • Exemplo:

    lista = [10, 20, 30]
matriz = [
    [1, 2, 3],
    [4, 5, 6]
]
print(lista[1])       # 20
print(matriz[0][2])  # 3
